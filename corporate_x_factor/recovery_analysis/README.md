# Recovery Analysis

Gatekeeper exposure assessments for judgments with `collectability_score >= 60`.

## Record Format

File naming: `recovery_<case_id>.json`

## Fields

- `gatekeeper_candidates` — list of potential gatekeepers
- `insurance_exposure` — estimated insurance exposure amount / carrier
- `settlement_viability_score` — 0–100

## Potential Gatekeepers

- Auditors
- Insurers
- Financial institutions
- Escrow agents
- Professional advisors

## Recovery Strategy Principles

1. **Settlement leverage first** — litigation is the last resort
2. Identify gatekeeper liability exposure before any outreach
3. Match to attorney network based on practice area fit
4. Coordinate plaintiff outreach only after attorney match confirmed
